# 紫微斗数排盘流程记录

**三合派为骨架 + 钦天四化为触发机制**，不混飞星、河洛、奇门、八字断语、塔罗类比。


## 0. 核心原则
紫微斗数排盘分两层：

第一层是**盘面生成**，也就是把出生信息换算成命盘结构：命宫、身宫、十二宫、五行局、主星、辅星、四化、大限、流年。

第二层是**盘面解读**，也就是看哪些宫位被触发，哪些结构形成倾向，哪些时间段风险较高或机会较强。

正式给你断盘时，我必须先完成排盘展示，再进入解读。不能一上来就说“你这个命如何如何”，那是跳步骤。

---

## 1. 输入信息

### 1.1 必需输入

做完整紫微斗数排盘，至少需要：

1. 性别
2. 出生年月日
3. 精确出生时间，最好到分钟
4. 出生地，至少城市/区
5. 出生地经度
6. 时区，默认中国标准时间 UTC+8，除非另说

如果没有经度，我会用城市中心经度近似，但必须标注：

> 经度采用城市中心近似，真太阳时可能存在数分钟误差。

### 1.2 高敏感信息

紫微斗数对这些变量尤其敏感：

1. 出生时间是否接近两个时辰交界
   例如 22:50、23:10、00:55、01:05 这种。

2. 真太阳时校正后是否跨时辰
   如果钟表时间是 23:05，但真太阳时可能变成 22:40，那么命宫、身宫、时柱都可能变。

3. 出生日期是否在节气交界附近
   这会影响节气四柱，尤其年柱、月柱。

4. 农历是否涉及闰月
   紫微斗数排星依赖农历月日，所以闰月要明确处理。

## 2. 时间换算

### 2.1 钟表时间

用户给出的出生时间通常是钟表时间，例如：

> 1998-10-17 14:30，出生地：安徽蚌埠，UTC+8

这只是原始输入，不直接用于最终盘面判断。

### 2.2 真太阳时 True Solar Time

真太阳时用于校正出生地经度带来的时间偏差。

中国标准时间以东经 120° 为标准经线。

公式：

```text
经度校正分钟 = (出生地经度 - 120°E) × 4 分钟
```

如果出生地在 120°E 以西，则真太阳时早于钟表时间。
如果出生地在 120°E 以东，则真太阳时晚于钟表时间。

例如出生地经度约 117.4°E：

```text
(117.4 - 120) × 4 = -10.4 分钟
```

也就是说：

```text
真太阳时 ≈ 钟表时间 - 10 分钟
```

### 2.3 均时差 EoT

理论上还可以加入 Equation of Time，也就是均时差。

完整公式：

```text
真太阳时 = 钟表时间 + 经度校正 + 均时差
```

但如果我无法可靠计算当天均时差，我会只做经度校正，并标注：

> 真太阳时已按经度校正，未计 EoT，属于近似真太阳时。

### 2.4 时辰判定

紫微斗数使用十二时辰：

```text
子：23:00–00:59
丑：01:00–02:59
寅：03:00–04:59
卯：05:00–06:59
辰：07:00–08:59
巳：09:00–10:59
午：11:00–12:59
未：13:00–14:59
申：15:00–16:59
酉：17:00–18:59
戌：19:00–20:59
亥：21:00–22:59
```

最终以**真太阳时对应的时辰**为准。

如果钟表时间和真太阳时跨了时辰，我必须明确提示：

> 此盘对出生时间高度敏感，需要校验现实经历，否则不能直接定盘。

## 3. 历法换算

紫微斗数排盘同时涉及两套历法信息：

一套是**节气四柱**，用于辅助结构校验。
一套是**农历年月日时**，用于安命宫、身宫、主星和辅星。

这两套不能混。

### 3.1 节气四柱

我会按节气口径计算：

1. 年柱：以立春为年界
2. 月柱：按节令划分，不按农历月份
3. 日柱：按干支纪日
4. 时柱：按真太阳时对应时辰起时干

月柱规则大致是：

```text
寅月：立春后
卯月：惊蛰后
辰月：清明后
巳月：立夏后
午月：芒种后
未月：小暑后
申月：立秋后
酉月：白露后
戌月：寒露后
亥月：立冬后
子月：大雪后
丑月：小寒后
```

注意：这一步主要用于结构校验，不替代紫微斗数本身。

### 3.2 农历年月日

紫微斗数安星主要依赖：

```text
农历出生月
农历出生日
出生时辰
```

所以需要将公历生日换算为农历日期。

如果遇到闰月，必须明确：

1. 是否闰月出生
2. 使用哪一种闰月处理口径
3. 是否与外部排盘软件一致

我的默认处理是：

> 以标准农历转换结果为准，闰月按实际农历月份记录；若用户提供文墨天机、紫微排盘软件结果，我会优先复核并沿用其农历转换结果。

## 4. 十二宫基本架构

### 4.1 固定地支宫位

紫微盘以十二地支为宫位框架：

```text
寅、卯、辰、巳、午、未、申、酉、戌、亥、子、丑
```

这十二宫是盘面的空间坐标。

### 4.2 定命宫

命宫用农历出生月与出生时辰确定。

规则：

```text
从寅宫起正月，顺数到出生农历月；
再从该宫起子时，逆数到出生时辰；
落点为命宫。
```

形式化写法：

```text
宫位顺序 = 寅 → 卯 → 辰 → 巳 → 午 → 未 → 申 → 酉 → 戌 → 亥 → 子 → 丑

base = 从寅宫按农历月顺数
命宫 = base 按出生时辰逆数
```

### 4.3 定身宫

身宫也用农历出生月与出生时辰确定，但方向不同。

规则：

```text
从寅宫起正月，顺数到出生农历月；
再从该宫起子时，顺数到出生时辰；
落点为身宫。
```

所以：

```text
命宫：月宫之后逆数时辰
身宫：月宫之后顺数时辰
```

命宫更像结构底盘，身宫更像后天用力位置。
正式解读时，命宫和身宫必须一起看。

### 4.4 十二人事宫布置

从命宫开始，逆布十二宫：

```text
命宫
兄弟宫
夫妻宫
子女宫
财帛宫
疾厄宫
迁移宫
交友宫 / 仆役宫
官禄宫
田宅宫
福德宫
父母宫
```

这一步会形成完整十二宫人事坐标。

需要根据出生年天干起寅宫天干。

```text
乙庚年：戊寅起
丙辛年：庚寅起
丁壬年：壬寅起
戊癸年：甲寅起
```

然后顺布十二宫天干。

例如如果是戊年或癸年：

```text
寅宫甲
卯宫乙
辰宫丙
巳宫丁
午宫戊
未宫己
申宫庚
酉宫辛
戌宫壬
亥宫癸
子宫甲
丑宫乙
```

### 5.2 定五行局

五行局由命宫干支纳音决定。

最终只会落在五类：

```text
水二局
木三局
金四局
土五局
火六局
```

五行局非常关键，因为它决定紫微星安放位置，也决定大限起始年龄。

## 6. 命主与身主

### 6.1 命主

命主由命宫地支决定。

常用表：

```text
子：贪狼
丑：巨门
寅：禄存
卯：文曲
辰：廉贞
巳：武曲
午：破军
未：武曲
申：廉贞
酉：文曲
戌：禄存
亥：巨门
```

命主更偏先天气质与命宫主题。

### 6.2 身主

身主由出生年支决定。

常用表：

```text
子、午：火星
丑、未：天相
寅、申：天梁
卯、酉：天同
辰、戌：文昌
巳、亥：天机
```

身主更偏后天行动方式和身体化的执行倾向。

正式输出时必须列出：

```text
命主：X
身主：Y
命宫：X宫
身宫：Y宫
```

## 7. 安十四主星

十四主星是紫微斗数核心，不是神煞堆砌。

### 7.1 十四主星列表

十四主星分两组：

紫微系：

```text
紫微
天机
太阳
武曲
天同
廉贞
```

天府系：

```text
天府
太阴
贪狼
巨门
天相
天梁
七杀
破军
```

### 7.2 先安紫微星

紫微星位置由两个变量决定：

```text
五行局
农历出生日
```

实际排盘时，我使用固定安紫微表或等价算法，不临场凭感觉推。

这一步输出：

```text
紫微星落于某宫
```

### 7.3 安紫微系诸星

紫微星确定后，按固定星曜布列规则安：

```text
紫微
天机
太阳
武曲
天同
廉贞
```

紫微系的相对位置是固定的。

### 7.4 安天府星

天府星位置由紫微星位置推得。

紫微与天府之间有固定对应关系。
天府确定后，再顺布天府系：

```text
天府
太阴
贪狼
巨门
天相
天梁
七杀
破军
```

### 7.5 主星输出标准

正式排盘时，我不会只说“命宫很好”这种废话，而是先给结构表：

```text
命宫：主星 X、Y
兄弟宫：主星 X
夫妻宫：主星 X、Y
……
父母宫：主星 X
```

如果某宫没有十四主星，要标注为空宫，并借对宫与三方四正来看，不直接说“空宫就没有”。

## 8. 安重要辅星

我只把重要辅星作为辅助结构，不让神煞喧宾夺主。

核心辅星包括：

```text
左辅、右弼
文昌、文曲
天魁、天钺
禄存
擎羊、陀罗
火星、铃星
地空、地劫
天马
红鸾、天喜
天刑、天姚
```

### 8.1 左辅右弼

左辅、右弼主要看月份。

常用规则：

```text
左辅：正月起辰，顺数至出生月
右弼：正月起戌，逆数至出生月
```

### 8.2 文昌文曲

文昌、文曲主要看出生时辰。

常用规则：

```text
文昌：按时辰安
文曲：按时辰安
```

它们关系到文书、表达、学历、系统化能力、审美与技术表达。

### 8.3 禄存

禄存按出生年天干安。

常用表：

```text
甲：寅
乙：卯
丙、戊：巳
丁、己：午
庚：申
辛：酉
壬：亥
癸：子
```

### 8.4 擎羊陀罗

擎羊、陀罗围绕禄存前后布置。

一般理解：

```text
擎羊：禄前
陀罗：禄后
```

它们代表阻力、硬碰硬、身体损耗、竞争压力、制度性冲突。

### 8.5 火星铃星

火星、铃星与出生年支、出生时辰相关。

这两个属于煞曜，判断时不能单独定吉凶，必须看：

```text
是否入命身
是否冲官禄、财帛、迁移
是否与七杀、破军、廉贞、贪狼等形成强触发
是否被禄权科化解
是否叠忌
```

### 8.6 天魁天钺

天魁、天钺按年干安，偏贵人、资质、关键节点助力。

但不能简单说“有贵人”。
要看它们落在哪个宫：

```text
命身：个人资质
官禄：职业机会
财帛：资源入口
迁移：外部机会
父母：上级/制度/长辈资源
田宅：资产与居住环境助力
```

## 9. 钦天四化

这是我的紫微模型里最重要的触发机制之一。

### 9.1 四化是什么

四化为：

```text
化禄
化权
化科
化忌
```

基本含义：

```text
禄：资源、欲望、收益、吸引力
权：控制、推动、压力、权责
科：名誉、秩序、缓冲、文书化
忌：执念、阻塞、债务、反复、代价
```

注意：化忌不是“必坏”，它更像系统里的高摩擦点和高执念点。

### 9.2 固定四化表

我的模型使用固定钦天四化表，不临时切换版本。

```text
甲：廉贞化禄，破军化权，武曲化科，太阳化忌
乙：天机化禄，天梁化权，紫微化科，太阴化忌
丙：天同化禄，天机化权，文昌化科，廉贞化忌
丁：太阴化禄，天同化权，天机化科，巨门化忌
戊：贪狼化禄，太阴化权，右弼化科，天机化忌
己：武曲化禄，贪狼化权，天梁化科，文曲化忌
庚：太阳化禄，武曲化权，太阴化科，天同化忌
辛：巨门化禄，太阳化权，文曲化科，文昌化忌
壬：天梁化禄，紫微化权，左辅化科，武曲化忌
癸：破军化禄，巨门化权，太阴化科，贪狼化忌
```

如果某外部排盘软件采用不同四化表，我不会混用。
要么以我的固定表重排，要么明确说明“沿用用户给定排盘结果”。

### 9.3 三层四化

解读时至少分三层：

```text
生年四化：原局长期结构
大限四化：十年阶段触发
流年四化：某一年触发
```

优先级通常是：

```text
生年结构 > 大限触发 > 流年触发
```

但如果流年四化与大限四化、生年四化叠到同一宫位，影响会明显放大。

## 10. 大限排布

### 10.1 大限顺逆

大限方向由性别与出生年天干阴阳决定。

规则：

```text
阳男阴女顺行
阴男阳女逆行
```

阳干：

```text
甲、丙、戊、庚、壬
```

阴干：

```text
乙、丁、己、辛、癸
```

例如：

```text
男命 + 阳年干 = 顺行
男命 + 阴年干 = 逆行
女命 + 阳年干 = 逆行
女命 + 阴年干 = 顺行
```

### 10.2 大限起始年龄

大限起始年龄由五行局决定：

```text
水二局：2岁起
木三局：3岁起
金四局：4岁起
土五局：5岁起
火六局：6岁起
```

每一大限十年。

例如水二局：

```text
2–11岁
12–21岁
22–31岁
32–41岁
……
```

如果是火六局：

```text
6–15岁
16–25岁
26–35岁
36–45岁
……
```

正式断盘时，我必须列出大限表，而不是只说“你现在走某某大运”。

## 11. 流年排布

流年主要看：

```text
流年天干四化
流年地支所在宫位
流年命宫
流年与当前大限重叠关系
```

例如分析 2026 年，就要看：

```text
2026 丙午年
丙干四化：
天同化禄
天机化权
文昌化科
廉贞化忌
```

然后看这些星在原局落在哪些宫，是否触发：

```text
命宫
身宫
官禄宫
财帛宫
迁移宫
田宅宫
夫妻宫
疾厄宫
```

如果流年忌进入大限重点宫位，或者冲动命身、官财迁，影响会更明显。

## 12. 三方四正分析

三合派核心不是单宫孤看，而是看三方四正。

每一个宫位都要看：

```text
本宫
对宫
三合宫一
三合宫二
```

例如命宫的三方四正常看：

```text
命宫
财帛宫
官禄宫
迁移宫
```

这四个宫共同构成人的行动结构：

```text
命宫：自我结构
财帛：资源与变现方式
官禄：事业与角色
迁移：外部环境与舞台
```

所以如果命宫一般，但官禄、财帛、迁移强，未必弱。
如果命宫看似强，但三方四正被忌煞冲破，也不能草率断强。

## 13. 解读优先级

正式解读时，我按这个顺序：

### 13.1 先看命身

```text
命宫主星
身宫位置
命主
身主
命身三方四正
```

判断一个人的底层结构：

```text
主动型 / 被动型
资源型 / 技术型 / 权力型 / 表达型
内控型 / 外部舞台型
稳定型 / 破局型
```

### 13.2 再看关键宫位

根据问题选择宫位。

事业：

```text
命宫
官禄宫
财帛宫
迁移宫
交友宫
父母宫
```

资产/房产：

```text
田宅宫
财帛宫
命宫
迁移宫
父母宫
福德宫
```

感情：

```text
夫妻宫
命宫
福德宫
迁移宫
子女宫
```

健康：

```text
疾厄宫
命身
福德宫
迁移宫
```

家族/父母/上级：

```text
父母宫
田宅宫
官禄宫
命宫
```

### 13.3 再看四化触发

判断重点不是“某宫有吉星所以好”，而是看四化如何流动：

```text
禄落哪里：资源入口在哪里
权落哪里：压力和控制权在哪里
科落哪里：缓冲、名誉、秩序在哪里
忌落哪里：代价、执念、卡点在哪里
```

尤其重点看：

```text
生年忌
大限忌
流年忌
忌是否冲命身
忌是否进入官财迁田夫疾等关键宫
禄忌是否同线
权忌是否同线
科是否能缓冲忌
```

### 13.4 最后看辅星与煞曜

辅星与煞曜不单独定性。

例如：

```text
擎羊入官禄
```

不能直接说事业差。
它可能表示：

```text
竞争强
行业硬
制度摩擦高
适合高压高门槛路径
容易与上级/规则硬碰硬
```

要结合主星、四化、大限流年判断。

---

## 14. 影响程度分级

我给影响程度时固定分三级：

```text
高
中
低
```

### 14.1 高影响

满足下列情况之一，通常定为高：

```text
生年忌、大限忌、流年忌叠加
命身、官禄、财帛、迁移、田宅、夫妻、疾厄被直接触发
大限宫与流年宫形成强冲
忌煞集中且无禄科缓冲
主星本身为强变动结构，如杀破狼，再遇权忌触发
```

### 14.2 中影响

通常是：

```text
单层四化触发
宫位有关但不是核心宫
有煞但同时有禄科缓冲
有机会也有摩擦
需要行动选择来决定结果
```

### 14.3 低影响

通常是：

```text
只触及弱相关宫位
只有辅星小触发
无明显四化叠加
与当前大限主线关系不强
```

## 15. 时间范围输出规则

紫微斗数不输出精确到某天某小时的断语。

我只能给区间，例如：

```text
2026–2027
2026 上半年
26–35岁大限
某大限前半段 / 后半段
某一年内较强
```

如果需要更细，只能说：

```text
上半年偏强
下半年偏回收
年中之后压力上升
年底前后有调整感
```

不能编造：

```text
2026年5月23日下午会发生某事
```

这种不做。

## 16. 正式输出模板

当你让我用紫微斗数分析某一段人生、某一年、某个问题时，我会按以下格式输出。

### 16.1 先展示计算结果

必须先列：

```text
1. 真太阳时
2. 节气四柱
3. 农历生日
4. 命宫、身宫
5. 五行局
6. 命主、身主
7. 十二宫星曜分布
8. 生年四化
9. 大限表
10. 本次分析涉及的流年四化
```

### 16.2 再进入解读

解读顺序：

```text
一、总纲
二、阶段化分析
三、关键事件候选
四、风险点
五、行动方案
六、现实校验点
```

### 16.3 每条判断必须包含

每条关键判断必须有：

```text
时间范围
影响程度
证据链
解决方案
```

例如：

```text
时间范围：2026–2027
影响程度：中高
证据链：流年廉贞化忌触发原局官禄三方，同时大限财帛宫被权忌牵动
解决方案：减少高杠杆扩张，保留现金缓冲，把合同、税务、股权边界前置处理
```

不能只说：

```text
你这两年要小心。
```

这种信息密度太低。

## 17. 解读时的证据链标准

每一个结论都要能追溯到盘面。

合格证据链：

```text
宫位 → 主星 → 三方四正 → 四化 → 大限/流年触发 → 现实含义
```

例如：

```text
官禄宫主星结构偏破局；
三方财帛宫有权星推动；
迁移宫被流年忌触发；
说明事业机会来自外部环境，但推进过程有制度/合作摩擦；
所以这段时间适合主动争取资源，但不适合裸奔式扩张。
```

不合格证据链：

```text
你命里有贵人。
```

这句话没有盘面路径，不算有效判断。

## 18. 校验机制

紫微斗数排盘最怕出生时辰误差。
所以我会在解读后给校验点，让你对照现实。

校验点通常包括：

```text
1. 童年家庭结构与父母关系
2. 早年迁移/换环境频率
3. 学业路径是稳定型还是跳跃型
4. 事业启动方式是平台型、资源型、技术型还是破局型
5. 财务模式偏现金流、资产、交易、股权还是管理费
6. 感情关系里是控制权问题、距离问题、节奏问题还是信任问题
7. 过去某个大限是否出现明显迁移、职业变化、资产变化
```

如果校验点明显不吻合，我会优先怀疑：

```text
出生时辰
真太阳时跨界
农历闰月处理
外部排盘软件口径差异
```

而不是硬拗解释。
